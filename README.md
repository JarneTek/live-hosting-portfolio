# Portfolio Project

## Installation Guide

### Setup Instructions

1. Install dependencies

```bash
npm install
```

2. Development commands

```bash

npm run start

npm run watch

npm run build
```

### Project Structure

- `src/scripts/` - JavaScript files
- `src/styles/` - CSS files
- `build/` - Compiled assets
